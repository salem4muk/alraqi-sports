# Alraqi Sports

Next.js app for Alraqi Sports, ready for deployment on DigitalOcean.

## Production Structure

- `app/` - Next.js pages and API routes.
- `public/` - static frontend assets, icons, images, CSS, and browser JS.
- `Dockerfile` - production Docker image for DigitalOcean Droplet or Container Registry.
- `archive/` - old/static builds and MikroTik package, kept out of Docker.
- `docs/` - project notes and design references, kept out of Docker.

## Local Development

```bash
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:4174/index.html
```

## Production Build

```bash
npm ci
npm run build
npm start
```

`npm start` runs the standalone Next.js server generated by `npm run build`.

## DigitalOcean App Platform

Use these settings:

- Build command: `npm ci && npm run build`
- Run command: `npm start`
- HTTP port: `3000`
- Node version: `22`

The app binds to `0.0.0.0` through the standalone Next.js server.

## Docker

Build:

```bash
docker build -t alraqi-sports .
```

Run:

```bash
docker run -p 3000:3000 alraqi-sports
```

Open:

```text
http://SERVER_IP:3000/index.html
```

## Important Notes

- The app fetches World Cup data through server-side API routes in `app/api/worldcup`.
- Channel links are configured in `app/api/channels/route.js`.
- Static assets and the PWA icon live in `public/assets`.
